# IoT Water Level Management System (Year 1 Semester 1 – Group Project)

## Overview
This project implements a water level management system using IoT concepts and Arduino. The system monitors water levels in a tank using sensors and provides automated control.

## Tools & Technologies
- **Arduino** – microcontroller for automation  
- **Sensors** – water level detection  
- **IoT Concepts** – data monitoring and alerts  

## Project Contents
- **Report:** detailed project description, diagrams, and implementation steps.  
- **Images:** Setup photos.  
- **Code:** Arduino code (`water_level_system.ino`).

## Notes
- This project is a Year 1 Semester 1 group assignment.  
- It demonstrates practical use of IoT in water management.
